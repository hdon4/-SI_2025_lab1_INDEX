За секој issue, првин започнав со git checkout на гранката каде што треба да работам. 
Правев промени во .java фајлот додавајки ги функциите за тој issue.
Откако ќе се имплементира функцијата, со командата git add, ги додадов новите промени.
После git add, git commit со параметар -m (за порака)
На крај, ја синхронизирав локалната гранка со remote гранката со git push origin IMENAGRANKA
Откако завршив со првите два issues, после на GitHub направив Pull and merge request на master
Откако ги завршив request-ите, so git pull ги зедов промените во master.
Ги спојв локално master гранката и последната гранка со git merge
И накрај синхронизирав со git push -u origin master
